# Polymarket 交易脚本控制台

一个面向 Polymarket 预测市场的自动化交易机器人**控制台界面设计稿**（暗色金融科技风格），用于实时监控运行中的策略脚本、成交、持仓与执行概览。

## 功能模块

- **侧边栏导航**：概览 / 市场 / 策略脚本 / 持仓 / 成交 / 日志 / 设置，底部含机器人状态卡（运行中、链上延迟等）
- **顶部状态栏**：页面标题、搜索、USDC 余额、新建脚本入口
- **KPI 指标**：总盈亏、持仓数量、24h 成交量、周脚本收益
- **运行中的策略脚本**：策略名称、触发规则、扫描频率、实时盈亏
- **监控的市场**：市场、YES 价格、24h 成交量、脚本动作
- **当前持仓**：市场、方向（YES/NO）、持仓价值、浮盈
- **最近成交**：时间、市场、买卖方向、金额

## 设计说明

- 强调色采用 Polymarket 紫罗兰 `#7C6CFF`，底色近黑 `#0A0A0C`
- 中文用 Noto Sans SC，英文与数字用 Geist / Geist Mono
- 盈亏配色遵循中文金融惯例：**盈利红色、亏损绿色**

## 技术栈（机器人侧）

- **接入**：Polymarket CLOB API + Gamma API，Polygon POS 链，USDC 结算
- **语言/SDK**：Node.js + TypeScript + `@polymarket/clob-client`（或 Python + `py_clob_client`）
- **策略/调度**：行情轮询 + 指标计算 + 调度器（cron / 独立进程）
- **基础设施**：托管平台（Railway / Render / 云）运行
